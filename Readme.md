## Task Making App ##

## A real-time task management application built using React for the frontend and Express.js with WebSocket for the backend.




## Tech Stack

### Frontend
- **React**  
- **React Router**  
- **Axios**  


### Backend
- **Node.js with Express**  
- **WebSocket** 
- **MongoDB** 

## For running Frontend
*** cd client
*** npm run start
